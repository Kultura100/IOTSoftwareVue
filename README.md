# IOTSoftware

Przedstawiony projekt zrealizowany został na poczet pracy Inżynierskiej.
Łączy ze sobą zagdnienia związane z elektroniką a także programowaniem webowym z użyciem najnowszego stosu technologicznego.
Przedstawiony projekt umożliwia podpięcie monitoringu IP z użyciem protokołu niewspieranego przez strony RTSP, integrację z różnymi czujnikami m.in. temperatury wewnątrz i na zewnątrz budynku oraz tworzenie raportów w oparciu o analizę danych pochodzących z czujników. 

Zrealizowany za pomocą framework'a Laravel z użyciem Vue i pakietu wspomagającego routing Intertia.js.
